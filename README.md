# Potion Aura Indicator

A Minecraft mod for **1.8.9** that displays active potion effects as highly visible particle rings floating above **other players' heads**. Each effect is represented by a distinct color ring, allowing you to identify an opponent's buffs and debuffs from a distance at a quick glance. This mod is especially helpful for players with visual impairments — like me — who want to clearly see what effects others are carrying in PvP fights.

---

# 药水效果光环指示器

一个用于 **1.8.9** 的 Minecraft 模组，将玩家身上的药水效果以**高亮粒子光圈**的形式显示在头顶，**站在远处也能清晰看到其他玩家**身上有哪些效果。每种效果对应一种独特的颜色，让你在 PvP 中快速判断对手的状态，无需靠近或打开背包。这个模组特别适合像我这样**视力不太好的玩家**，让战斗信息一目了然。****
